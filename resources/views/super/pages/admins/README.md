# Direktori Pengelola Barang

Folder ini masih bernama `admins` karena mengikuti route teknis `super.admins.*` dan controller `AdminDirectoryController`.

Isi halaman ini adalah direktori **Pengelola Barang** yang dilihat oleh Super Admin. Label tampilan, CSS frontend, dan README sudah memakai istilah pengelola barang/manager.

Jangan rename folder ini sendiri sebelum route, controller, test, dan semua pemanggilan `view('super.pages.admins.index')` ikut dirapikan dalam tahap teknis berikutnya.
